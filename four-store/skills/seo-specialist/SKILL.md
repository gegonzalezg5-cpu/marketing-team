4. SEO Local — Chile
Google Business Profile:

Nombre: Four Store
Categoría: Tienda de artículos deportivos / Proveedor de uniformes
Descripción: incluir "uniformes sublimados", "entrega 3-5 días", "35 años de experiencia"
Publicar fotos de equipos reales vestidos con Four Store (UGC)
Responder reseñas mencionando keywords naturalmente

Señales locales clave:

Mencionar "Chile" y ciudades principales en contenido de páginas
Obtener menciones en sitios chilenos (federaciones, ligas, colegios)
Crear páginas específicas por región si hay volumen de búsqueda


5. Core Web Vitals
LCP — Largest Contentful Paint (objetivo: < 2.5s)

Comprimir imágenes de productos a formato WebP
Usar CDN para imágenes
Precargar imagen principal de cada página

CLS — Cumulative Layout Shift (objetivo: < 0.1)

Definir dimensiones fijas en todas las imágenes de producto
Reservar espacio para banners y sliders

FID — First Input Delay (objetivo: < 100ms)

Diferir scripts de terceros (chat, analytics) con defer o async


6. Estrategia de Contenido para Ecommerce sin Blog
Sin blog, el SEO se concentra en:
Páginas de categoría optimizadas:

Agregar 300-500 palabras descriptivas arriba o abajo del listado de productos
Incluir preguntas frecuentes (FAQ Schema) en cada categoría

Páginas de producto:

Descripciones únicas de 200-300 palabras (no copiar del proveedor)
Incluir beneficios orientados al cliente (club, colegio, empresa)
Agregar sección "¿Para quién es ideal este producto?"

Páginas de aterrizaje por segmento (recomendado crear):

/uniformes-clubes-futbol — para clubes amateur
/uniformes-colegios — para instituciones educativas
/uniformes-empresas — para equipos corporativos
/accesorios-por-deporte — para deportistas individuales


7. Checklist Antes de Publicar una Página

 Title tag con keyword principal (menos de 60 caracteres)
 Meta description llamativa con CTA (150-160 caracteres)
 H1 con keyword principal
 URL limpia y descriptiva
 Imágenes en WebP con alt text descriptivo
 2-3 enlaces internos a otras páginas relevantes
 Schema markup de producto implementado
 Mención de Chile y/o región si aplica
 Tiempo de carga menor a 3 segundos en móvil
 Canonical tag correcto


8. Métricas a Monitorear
Herramientas:

Google Search Console — rankings, clics, errores de indexación
Google Analytics 4 — tráfico orgánico, conversiones
PageSpeed Insights — Core Web Vitals
Ahrefs o SEMrush — keywords y competencia en Chile

KPIs clave para Four Store:

Posición promedio para keywords de uniformes en Chile
Tráfico orgánico mensual a páginas de producto
CTR desde Google (objetivo: > 3%)
Conversiones (cotizaciones o compras) desde tráfico orgánico


Prioriza siempre la experiencia del usuario. Google Chile recompensa el contenido útil, claro y de confianza. El slogan "Tu equipo, tu identidad" debe sentirse en cada página.
